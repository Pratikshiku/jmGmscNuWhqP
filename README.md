### 三、功能描述
系统分为（身份）：普通用户（游客）、导游、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 旅游景点
6. 酒店信息
7. 留言板
8. 公告信息
9. 个人中心
10. 旅游景点收藏
11. 酒店收藏
12. 酒店预定


**导游功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 旅游景点管理

7. 酒店信息管理
8. 酒店留言管理
9. 论坛管理
10. 公告信息管理



**管理员功能：**
1. 登录
